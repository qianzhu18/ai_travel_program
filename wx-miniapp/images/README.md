# 图标资源目录

## 需要准备的图标

### TabBar图标 (81x81px)

- `home.png` - 首页图标(未选中状态)
- `home-active.png` - 首页图标(选中状态)
- `user.png` - 我的图标(未选中状态)
- `user-active.png` - 我的图标(选中状态)

### 其他图标

- `default-avatar.svg` - 默认头像 (120x120px)

## 临时方案

如果暂时没有图标,可以:

1. **注释掉tabBar配置**
   编辑 `app.json`,注释掉 `tabBar` 部分

2. **使用简单占位图**
   用纯色图片临时代替

3. **在线生成图标**
   使用 iconfont.cn 或其他图标网站生成

## 图标设计建议

- 风格: 线性图标或面性图标
- 颜色:
  - 未选中: #bcaea8 (灰色)
  - 选中: #e89a8d (粉色)
- 格式: PNG (支持透明背景)
- 尺寸: 按规范提供@2x和@3x版本
